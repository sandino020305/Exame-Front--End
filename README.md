# Sistema de Baralho

Projeto desenvolvido para exame de Front-end usando Next.js.

## Tecnologias Usadas

- Next.js 14
- React
- TypeScript
- CSS Modules

## Instruções

1. Instalar dependências:
```
npm install
```

2. Criar arquivo .env.local na raiz do projeto:
```
AUTH_PASSWORD=sua_senha
```

3. Executar o projeto:
```
npm run dev
```

4. Acessar ( sem link )
